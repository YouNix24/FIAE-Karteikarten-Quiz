JSON ist...
Java Script Object Notation
leichtes Datenaustauschformat
einfacher Text in JS - Syntax
ein Standard zum Senden von Daten

Ähnelt syntaktisch dem Code zum Erstellen von JS-Objekten
Integrierte Funktionen in JS:
JSON.parse() - Erstellt Objekte
JSON.stringify() - Erstellt JSON-Zeichenketten

Zeichen der Basisstruktur
{}[]:,

Syntax & Datentypen
String

“Berlin”

Unicode, in doppelten Anführungszeichen

Number

42 ; 3.1415 ; -13

Boolean

true ; false

Kein führendes 0, Punkt als
Dezimaltrenner
Kleinschreibung!

Null

null

Platzhalter für “kein Wert”

Array

[“rot”, “grün”, “blau”]

Geordnete Liste, 0-basiert

Object

{”stadt”:”Berlin”, “plz”:10115}

Ungeordnete Wert-Paare

Regeln bei JSON
Nur doppelte Anführungszeichen
Letztes Element in Arrays/Objects
ohne Komma
Schlüsselnamen immer als String
UTF-8-ohne-BOM empfohlen
